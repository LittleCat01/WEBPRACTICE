  - 类选择器，一般都用这个
  - 通配符选择器
- 高级选择器
  - 略
- 类选择器
  - 同一个标签可以使用多个类选择器。用空格隔开
  - 类选择器的使用，能够决定一个人的 css 水平
    - 正确的思路，就是用所谓“公共类”的思路，就是我们类就是提供“公共服务”，比如有绿、大、线，一旦携带这个类名，就有相应的样式变化
    - 然后让每个标签去选取自己想要用的类选择器
    - 不要去试图用一个类名，把某个标签的所有样式写完
    - 每一个类要尽可能小，有“公共”的概念，能够让更多的标签使用

## Flex 布局

CSS3 中的 flex 属性，在布局方面做了非常大的改进，使得我们对多个元素之间的布局排列变得十分灵活，适应性非常强。其强大的伸缩性和自适应性，在网页开中可以发挥极大的作用

- 使用
  - display：flex
  - flex 唯一的缺点就在于，它不支持低版本的 IE 浏览器
- 定义
  - 弹性盒子：声明父容器，给父容器加属性
  - 子元素/弹性元素：容器里的元素
  - 主轴：默认是水平方向，从左向右
  - 侧轴：与主轴垂直的轴称作侧轴，默认是垂直方向，从上往下
- 属性
  - 略

## 字体

开发人员可以为自已的网页指定特殊的字体（将指定字体提前下载到站点中），无需考虑用户电脑上是否安装了此特殊字体。从此，把特殊字体处理成图片的方式便成为了过去。

- 字体常见格式
- 为不同浏览器准备不同格式字体
  - 字体生成工具，网站
- 字体图标
  - 我们其实可以把图片制作成字体。常见的做法是：把网页中一些小的图标，借助工具生成一个字体包，然后就可以像使用文字一样使用图标了
  - 将所有图标打包成字体库，减少请求；
  - 具有矢量性，可保证清晰度；
  - 使用灵活，便于维护
